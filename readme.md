# 🎓 Student Score Prediction App

An interactive Machine Learning web application built with **Streamlit** that predicts a student's final exam score based on study hours using a **Linear Regression** model.

The application allows users to explore the dataset, visualize the regression line, evaluate model performance, and predict scores in real time through a simple and interactive interface.

---

## 🚀 Features

- 🎯 Predict student scores based on study hours
- 📊 Interactive Streamlit dashboard
- 📋 Dataset preview
- 📈 Regression line visualization
- 📉 Model evaluation metrics (MAE, MSE, RMSE & R² Score)
- ⚡ Real-time prediction
- 🎨 Clean and responsive user interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/2006181/student-score-prediction-app.git
```

Move into the project folder

```bash
cd student-score-prediction-app
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
